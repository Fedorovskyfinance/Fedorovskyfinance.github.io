# Praxis · Бренд-система

Источник правды для всех продающих материалов. Поменял токен здесь → поменялось во всех материалах.

## Файлы
- `tokens.css` — палитра (Light по умолчанию + Dark через `[data-theme="dark"]`), шрифты, шкала отступов. **Единственное место, где живут цвета.**
- `components.css` — классы-примитивы (`.kicker .lead .pill .cta .num .disclaimer .wordmark .slide .reveal`) + базовые `h1/h2` на Spectral. Только `var(--*)`, без сырых цветов.
- `deck.js` — контроллер слайд-деков (клавиатура ← →, точки навигации, прогресс-бар, появление по скроллу).

## Как подключить в новый материал
```html
<!-- Шрифты бренда: Spectral (заголовки) + Satoshi (текст) + Clash Display (цифры) -->
<link href="https://fonts.googleapis.com/css2?family=Spectral:ital,wght@0,400;0,500;0,600;1,400&display=swap" rel="stylesheet">
<link rel="stylesheet" href="https://api.fontshare.com/v2/css?f[]=clash-display@500,600,700&f[]=satoshi@400,500,700&display=swap">
<link rel="stylesheet" href="../brand/tokens.css">
<link rel="stylesheet" href="../brand/components.css">
```
В самом материале — только `var(--*)` и классы. Никаких `#rrggbb` в теле.

## Две темы
- **Light (по умолчанию)** — доверие, безопасность, рост. ~90% контента.
- **Dark** (`data-theme="dark"` на секции) — обложки, блок «почему безопасно», эксклюзив.

## Язык материалов — для врача, не для маркетолога
Клиентские материалы пишем простым языком. Запрещён жаргон: «лидген», «воронка», «прогрев», «E-E-A-T», «ROI», «KPI», «топ-3 Яндекса» и т.п. Каждое слово должно быть понятно занятому врачу без гугла. Подробно — `memory/plain-language-for-doctors.md`.

## Гарантия (формулировка)
Risk-reversal = **«Не сделаем обещанный объём работы за месяц — вернём деньги»**. Без «топ-3 Яндекса», без процентов с выручки врача.

## Экспорт в PDF (для отправки врачу)
Открыть материал в Chrome → Печать (Cmd+P) → «Сохранить как PDF». Фоновые цвета сохраняются (`-webkit-print-color-adjust:exact` в `components.css`).

## Проверка на штрафы (автотест)
Видимый текст всех материалов прогоняется через движок комплаенса (`praxis_engine.compliance.rules`):
```
python3 -m pytest tests/brand/ -v
```
Проверяет: нет запрещённых формулировок (ФЗ-38), дисклеймер в материалах с оффером, нет упоминаний VK, материал линкует токены и шрифты бренда.
